\section*{Визуальные среды разработки}

\begin{itemize}
  \item \textbf{MATLAB/Simulink} (http://www.mathworks.com/)
    Графический редактор для разработки встроенных систем. Поддерживает генерацию 
    в C++ и VHDL, использует нестандартную нотацию. Широкие возможности для расширения.
  \item \textbf{Artisan Studio} (http://www.artisansoftwaretools.com/)
    Графическая среда разработки, поддерживающая UML и SysML. Поддерживает совместную разработку
    --- разработчики модифицируют модели на сервере. Интеграция с Simulink, импорт моделей из Rational Rose.
  \item \textbf{dSPACE} (http://www.dspace.com)    
    Графический редактор для разработки встроеного ПО (в основном для автомобилей).
    Использует нестандартную нотацию, имеет генератор (C), симулятор, работает вместе с Simulink.
  \item \textbf{FalconML} (http://www.axilica.com/)  
    Графическая среда разработки аппаратного обеспечения. Поддерживает UML. Генерация в SystemC, симуляция.
  \item \textbf{Visual Elite} (http://www.mentor.com/products/esl/system\_integration/visual\_elite/)
    Графический редактор, нестандартная нотация, генерация в VHDL, Verilog и SystemC.      
  \item \textbf{Mentor Graphics HDL Designer} (http://www.mentor.com/products/fpga\_pld/hdl\_design/ hdl\_designer\_series/)
    Среда разработки для программирования FPGA. Визуальный редактор с нестандартной нотацией, генерация в VHDL, Verilog, SystemC и др.,
    посторение диаграмм по существующему коду. 
  \item \textbf{Aldec Active-HDL 8.1} (http://www.aldec.com/Products/Product.aspx?productid=ff724e4e-71f8-48db-bdf6-56f43fddf586)
    Интегрированная среда разработки для FPGA, возможность рисовать структурные диаграммы и диаграммы состояний,
    генерация диаграмм по коду. Нестандартная нотация. Генерация в VHDL, Verilog и EDIF. 
  \item \textbf{XModelink} (http://www.zipc.com/english/product/xmodelink/index.html)
    Графическая среда разработки, поддерживает профиль UML для SystemC. Генерация в SystemC, симуляция.
    Реализован как плагин к Eclipse.
  \item \textbf{Green Mountain VHDL Studio} (http://www.gmvhdl.com/VHDLStudio.html)
    Интегрированная среда разработки для VHDL, имеет встроенный редактор диаграмм состояний с генерацией VHDL.
    Последнее обновление --- 2001 год.
  \item \textbf{Altera Quartus II} (http://www.altera.com/products/software/quartus-ii/subscription-edition/qts-se-index.html)
    Графический редактор, генерация в VHDL и Verilog, нестандартная нотация. Средства интеграции с другими инструментами.
  \item \textbf{V2HDL: Visual VHDL} (http://www.cs.drexel.edu/\~bjb39/)
    Некоммерческий проект, создавался в целях обучения студентов VHDL. Позволяет рисовать устройства, 
    собирая их из набора логических элементов. Генерирует читабельный код на VHDL. Последнее обновление --- 2006 год.
  \item \textbf{VGUI} (http://www.atl.external.lmco.com/projects/rassp2/vgui/)
    Свободно распространяемый графический редактор, позволяющий генерировать структурный уровень VHDL или Verilog. Создание
    работоспособной системы требует ручной правки. Нотация нестандартная. Последнее обновление --- в 2003 году.

\end{itemize}
